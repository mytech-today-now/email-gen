## Cross-cutting evidence

- Accessibility baseline: semantic controls/labels, focus-visible styles, keyboard split panes, focus-trapped dialogs, ARIA live status, reduced motion/high contrast, axe, and responsive reflow target WCAG 2.2 AA. WCAG 3 remains a draft and is not claimed as the conformance standard.
- Security baseline: loopback, Origin/CSRF, CSP, bounded input, sanitizer, SSRF defenses, memory-only runtime credentials, redacted diagnostics, and archive hardening.
- Release gates: lint, formatting, 107-test Vitest suite, Playwright Chromium/Firefox/WebKit projects, coverage, and zero-vulnerability npm audit are recorded in the final verification run.
